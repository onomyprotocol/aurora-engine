* Build
```
docker build -t onomy/aurora-engine-local --progress=plain --network=host -f localnet.Dockerfile .
```

* Run
```
docker run -p 3030:3030 --name aurora-engine-local -it onomy/aurora-engine-local
```

* Login to the container
```
docker exec -it aurora-engine-local bash
```

* Examine engine:
```
export AURORA_ENGINE=aurora.node0
aurora get-version
aurora get-owner
aurora get-chain-id
aurora encode-address aurora.node0
aurora get-balance 0x177DF8c673ef3007eCa4b2BaBcFe2A6CaF529dd8
```

